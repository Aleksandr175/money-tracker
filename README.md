## Install

- Docker
    - ./vendor/bin/sail + command
      - ex. **/vendor/bin/sail composer require laravel/breeze --dev**
      - alias sail='[ -f sail ] && sh sail || sh vendor/bin/sail'
      - sail up
- Package json
  - script 
    - "build": "vite build"
  - "npm run watch" - watcher for building project
  - "npm run dev" - watcher with typescript
